## TelegramBot

## Deploy
	Config your username and token in EchoBot.java
	Build:	mvn clean install	
	Run:	mvn exec:java -Dexec.mainClass="Main"


## Using lib:
    <dependency>
        <groupId>org.telegram</groupId>
        <artifactId>telegrambots</artifactId>
        <version>3.5</version>
    </dependency>
